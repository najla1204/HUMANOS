## ðŸŽ¯ What is HUMANOS?

**HUMANOS** is an AI-powered decision simulation sandbox that helps you visualize and compare the outcomes of major life decisions. By creating a "digital twin" based on your core values and skills, it runs stochastic simulations to project how different paths might unfold.

> *"Decisions are permanent. Simulations aren't."*

## âœ¨ Features

- **Scenario Simulation** â€” Input two divergent life paths (e.g., corporate job vs. startup) and receive AI-generated trajectory projections
- **Digital Twin Profiling** â€” Define your identity, values, and skills to anchor personalized simulations
- **Radar Analysis** â€” Visual comparison of skill growth, value alignment, optionality, and friction across scenarios
- **Synthesis Path** â€” Automatically generates a third "hybrid" path that combines the best of both options
- **Journey Logs** â€” Save and revisit past simulations to track your decision-making evolution
- **Dual Engine Modes** â€” Choose between *Instant Flash* (fast) or *Stochastic Pro* (deep analysis)

##  Getting Started

### Prerequisites
- **Node.js** (v18+)
- **Gemini API Key** â€” Get one at (https://ai.google.dev/)

### Installation

```bash
# Clone the repository
git clone https://github.com/najla1204/HUMANOS.git
cd HUMANOS

# Install dependencies
npm install

# Configure environment
# Add your Gemini API key to .env.local
echo "GEMINI_API_KEY=your_api_key_here" > .env.local

# Run the app
npm run dev
```

The app will be available at `http://localhost:5173`
